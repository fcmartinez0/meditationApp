# Stillness

A calm, cross-platform meditation app — breathe, focus, and rest with a timer,
guided breathing, and a rich library of sounds. Built with Expo (React Native +
TypeScript); one codebase runs on iOS, Android, and the web.

## Features

- **Meditation timer** — pick a length, choose a sound, and follow a calming
  visual (breathing orb, a filling "tide," or a minimal clock).
- **Guided breathing** — Box, 4-7-8, Calm, and Coherent rhythms with an animated
  pace guide and haptics.
- **A big sound library**
  - **Ambient:** rain, ocean, forest, stream, campfire, night crickets, brown /
    pink / white noise, and a cat purr.
  - **Frequencies:** binaural-beat tracks (Calm, Clarity, Focus, Dream, Deep).
  - **Beats:** instrumental grooves, each an homage to an artist's style.
  - **Generative:** endless, never-repeating music that **learns what you like**
    when you rate it.
- **Streaks & progress** — daily streak, total minutes and sessions, weekly chart.
- **Daily reminders** — an optional gentle nudge at a time you choose.

All sounds are generated by the app itself — no recordings or third-party audio.

> Headphones are recommended for the binaural and low-frequency tracks. The
> frequency tracks are a wellness aid, not a medical treatment.

## Running it

```bash
npm install
npx expo start          # press i (iOS), a (Android), or scan the QR with Expo Go
```

For a full native build (Xcode / Android Studio required):

```bash
npx expo run:ios
npx expo run:android
```

Cloud builds and store submission use EAS (no Mac required):

```bash
npm i -g eas-cli && eas login
eas build -p ios --profile preview     # installable build via link
eas submit -p ios                       # TestFlight / App Store
```

## Privacy

Stillness is **local-first**: there's no account and no server. Your settings,
streak, history, and music preferences live only on your device, and there's no
tracking, ads, or analytics. A Privacy & Disclaimer page is available in the app
(Settings → About).
